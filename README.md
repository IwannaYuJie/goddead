# Goddead

Static archive site for [goddead.com](https://goddead.com).

## Current Experience

**空缺档案馆 (Absence Archive)** — a single-page, editorial reading experience after the death of God.

- Monumental `GOD / DEAD` entrance
- Nine lore chapters: prologue, doctrines, chronicle, forbidden index, map, witnesses, epithets, remains, reliquary
- Visible **目录** menu with section jump links
- Light interactions (signal awaken, archive accordion, map, visitor marks) without minigame subpages

## Deploy

This repository is intended to deploy through Cloudflare Pages.

- Build command: none
- Output directory: `/`
- Custom domain: `goddead.com`
- Pages URL: [goddead.pages.dev](https://goddead.pages.dev/)
